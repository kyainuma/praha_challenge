## add
### 正常系
- [ ] [3, 10, 3]を受け取った場合、16が返ること
- [ ] 30個の引数を受け取った場合、和が返ること
- [ ] 計算結果が1000を超える場合、「too big」と文字列が返ること

### 異常系
- [ ] 数字以外の引数を受け取った場合、エラーが発生すること
- [ ] 引数が0個だった場合、エラーが発生すること
- [ ] 31個以上の引数を受け取った場合、エラーが発生すること
